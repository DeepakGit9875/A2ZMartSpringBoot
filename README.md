A2ZMart – E-Commerce Backend Application
Overview

A2ZMart is a backend-driven e-commerce application developed using Java and Spring Boot. The project provides secure REST APIs for user authentication, product management, shopping cart, order processing, and payment workflows.

The backend is fully implemented and deployed on AWS using Amazon EC2 and Amazon RDS (PostgreSQL). The frontend is currently under development using React.
